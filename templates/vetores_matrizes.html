Vetores e Matrizes em Python

Embora Python não tenha tipos de dados chamados explicitamente "vetor" ou "matriz" como em algumas outras linguagens, as **listas** (list) são estruturas de dados incrivelmente versáteis que podem ser usadas para representar essas coleções de forma eficiente.

  • Um vetor (array unidimensional) é tipicamente implementado como uma lista simples.
  • Uma matriz (array bidimensional) é comumente implementada como uma lista de listas (onde cada lista interna representa uma linha da matriz).

Vamos explorar como trabalhar com essas estruturas.

Vetores (Usando Listas)

Um vetor pode ser visto como uma coleção ordenada de itens, onde cada item pode ser acessado por um índice numérico (posição). Em Python, uma lista serve perfeitamente a este propósito.

Criando um Vetor (Lista):
# Um vetor de números inteiros
numeros = [10, 20, 30, 40, 50]
print(numeros)

# Um vetor de strings
nomes = ["Ana", "Carlos", "Beatriz"]
print(nomes)

# Um vetor vazio
vetor_vazio = []
print(vetor_vazio)
Acessando Elementos:

Os elementos são acessados por seus índices, começando em 0 para o primeiro elemento.

numeros = [10, 20, 30, 40, 50]
print(f"Primeiro elemento: {numeros[0]}")  # Saída: 10
print(f"Terceiro elemento: {numeros[2]}")  # Saída: 30
print(f"Último elemento: {numeros[-1]}")   # Saída: 50 (indexação negativa)
Modificando Elementos:
nomes = ["Ana", "Carlos", "Beatriz"]
print(f"Lista original: {nomes}")
nomes[1] = "Carlos Alberto"  # Modifica o segundo elemento
print(f"Lista modificada: {nomes}")
Comprimento do Vetor:

A função len() retorna o número de itens em uma lista.

numeros = [10, 20, 30, 40]
print(f"O vetor tem {len(numeros)} elementos.") # Saída: 4
Iterando sobre um Vetor:

Você pode usar um loop for para percorrer todos os itens.

nomes = ["Ana", "Carlos", "Beatriz"]
for nome in nomes:
    print(f"Olá, {nome}!")
Adicionando e Removendo Elementos:
  • append(item): Adiciona um item ao final da lista.
  • insert(indice, item): Insere um item em uma posição específica.
  • pop(indice): Remove e retorna o item de uma posição (padrão: último).
  • remove(item): Remove a primeira ocorrência de um item específico.
cores = ["vermelho", "verde"]
print(f"Cores: {cores}")

cores.append("azul")      # Adiciona ao final
print(f"Após append: {cores}")

cores.insert(1, "amarelo") # Insere na posição 1
print(f"Após insert: {cores}")

cor_removida = cores.pop() # Remove o último ("azul")
print(f"Após pop: {cores}, Cor removida: {cor_removida}")

cores.remove("vermelho")  # Remove "vermelho"
print(f"Após remove: {cores}")

Matrizes (Usando Listas de Listas)

Uma matriz bidimensional pode ser visualizada como uma tabela com linhas e colunas. Em Python, implementamos isso como uma lista onde cada item é, ele próprio, uma lista representando uma linha.

Criando uma Matriz (Lista de Listas):
# Matriz 3x3 (3 linhas, 3 colunas)
matriz = [
    [1, 2, 3],  # Linha 0
    [4, 5, 6],  # Linha 1
    [7, 8, 9]   # Linha 2
]
print(matriz)

# Para visualizá-la de forma mais clara:
for linha in matriz:
    print(linha)
Acessando Elementos:

Para acessar um elemento, você precisa de dois índices: matriz[indice_da_linha][indice_da_coluna].

matriz = [ [1,2,3], [4,5,6], [7,8,9] ]

print(f"Elemento na linha 0, coluna 0: {matriz[0][0]}")  # Saída: 1
print(f"Elemento na linha 1, coluna 2: {matriz[1][2]}")  # Saída: 6
print(f"Elemento na linha 2, coluna 1: {matriz[2][1]}")  # Saída: 8
Modificando Elementos:
matriz = [ [1,2,3], [4,5,6], [7,8,9] ]
print(f"Matriz original (linha 1): {matriz[1]}")

matriz[1][1] = 55  # Modifica o elemento na linha 1, coluna 1
print(f"Matriz modificada (linha 1): {matriz[1]}")
# Matriz completa após modificação: [[1, 2, 3], [4, 55, 6], [7, 8, 9]]
Iterando sobre uma Matriz:

Você geralmente usará loops aninhados (um loop for dentro de outro) para percorrer todos os elementos.

matriz = [ [1,2,3], [4,5,6], [7,8,9] ]

# Iterando por linha, e depois por elemento em cada linha
print("Iterando elemento por elemento:")
for linha in matriz:
    for elemento in linha:
        print(elemento, end=" ") # end=" " para imprimir na mesma linha, separado por espaço
    print() # Pula uma linha após cada linha da matriz

# Acessando por índices (se precisar dos índices)
print("\nIterando com índices (linha, coluna):")
num_linhas = len(matriz)
num_colunas = len(matriz[0]) # Assume que todas as linhas têm o mesmo número de colunas

for i in range(num_linhas):
    for j in range(num_colunas):
        print(f"Matriz[{i}][{j}] = {matriz[i][j]}")

Bibliotecas para Operações Avançadas

Para operações matemáticas mais complexas e eficientes com vetores e matrizes (especialmente em áreas como ciência de dados, aprendizado de máquina e computação científica), a biblioteca NumPy é amplamente utilizada em Python. Ela oferece estruturas de array otimizadas e uma vasta gama de funções.

No entanto, para muitos propósitos e para aprender os conceitos fundamentais, as listas do Python são perfeitamente adequadas e muito flexíveis!
