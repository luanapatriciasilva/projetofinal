Estruturas de Seleção em Python

Em Python, como em muitas outras linguagens de programação, as estruturas de seleção (ou estruturas condicionais) são cruciais para controlar o fluxo de execução do código. Elas permitem que seu programa tome decisões com base em certas condições, executando blocos de código específicos apenas se essas condições forem verdadeiras (ou falsas, conforme a lógica).

As principais palavras-chave para estruturas de seleção em Python são: if, elif (else if), e else.

O Comando if

A estrutura de seleção mais fundamental é o comando if. Ele avalia uma condição booleana (uma expressão que resulta em True ou False) e, se a condição for verdadeira, o bloco de código indentado logo abaixo dele é executado.

Sintaxe:
if condicao:
    # Bloco de código a ser executado se a condicao for True
    # Este bloco DEVE ser indentado (geralmente 4 espaços)
    print("A condição é verdadeira!")
Exemplo Prático:
idade = 20
if idade >= 18:
    print("Você é maior de idade.")
    print("Pode entrar na festa!")

Neste exemplo, como a variável idade (20) é maior ou igual a 18, ambas as mensagens dentro do bloco if serão impressas.

O Comando else

Muitas vezes, queremos que um bloco de código diferente seja executado caso a condição do if seja falsa. Para isso, utilizamos o comando else, que é opcional e sempre vem após um bloco if (ou elif).

Sintaxe:
if condicao:
    # Bloco de código se a condicao for True
    print("Condição verdadeira.")
else:
    # Bloco de código se a condicao for False
    print("Condição falsa.")
Exemplo Prático:
temperatura = 15
if temperatura > 25:
    print("Está calor! Vamos à praia.")
else:
    print("Não está tão calor. Melhor levar um casaco.")

Aqui, como temperatura (15) não é maior que 25, o bloco de código dentro do else será executado.

O Comando elif (Else If)

Quando precisamos testar múltiplas condições em sequência, podemos usar o comando elif (uma contração de "else if"). O Python avalia as condições do elif apenas se as condições anteriores (do if ou de outros elifs) forem falsas. Você pode ter quantos blocos elif forem necessários. O bloco else, se presente, será executado se nenhuma das condições anteriores (if ou elif) for verdadeira.

Sintaxe:
if condicao1:
    # Bloco de código se condicao1 for True
    print("Condição 1 atendida.")
elif condicao2:
    # Bloco de código se condicao1 for False e condicao2 for True
    print("Condição 2 atendida.")
elif condicao3:
    # Bloco de código se condicao1 e condicao2 forem False e condicao3 for True
    print("Condição 3 atendida.")
else:
    # Bloco de código se NENHUMA das condições anteriores for True
    print("Nenhuma das condições principais foi atendida.")
Exemplo Prático:
nota = 75
if nota >= 90:
    print("Conceito: A")
elif nota >= 80:
    print("Conceito: B")
elif nota >= 70:
    print("Conceito: C")
elif nota >= 60:
    print("Conceito: D")
else:
    print("Conceito: F (Reprovado)")

Neste caso, "Conceito: C" será impresso, pois é a primeira condição (de cima para baixo, após as anteriores serem falsas) que se torna verdadeira.

Operadores de Comparação e Lógicos

As condições nas estruturas de seleção são geralmente formadas usando operadores de comparação e operadores lógicos.

Operadores de Comparação Comuns:
  • == Igual a
  • != Diferente de
  • < Menor que
  • > Maior que
  • <= Menor ou igual a
  • >= Maior ou igual a
Operadores Lógicos Comuns:
  • and E lógico (ambas as condições devem ser verdadeiras)
  • or OU lógico (pelo menos uma condição deve ser verdadeira)
  • not NÃO lógico (inverte o valor booleano da condição)
Exemplo com Operadores Lógicos:
idade = 22
possui_carteira_motorista = True

if idade >= 18 and possui_carteira_motorista:
    print("Pode dirigir legalmente.")
else:
    print("Não pode dirigir legalmente.")

tem_sol = False
fim_de_semana = True

if tem_sol or fim_de_semana:
    print("Bom momento para um passeio!")

Estruturas if Aninhadas

Você pode colocar uma estrutura if/elif/else dentro de outra. Isso é chamado de aninhamento. Use com moderação, pois pode tornar o código mais difícil de ler se o aninhamento for muito profundo.

Exemplo Prático:
idade = 25
nacionalidade = "Brasileira"

if idade >= 18:
    print("Maior de idade.")
    if nacionalidade == "Brasileira":
        print("Pode votar no Brasil.")
    else:
        print("Estrangeiro maior de idade.")
else:
    print("Menor de idade.")

Expressão Condicional (Operador Ternário)

Python oferece uma forma concisa de escrever um if/else simples em uma única linha, conhecida como expressão condicional ou operador ternário.

Sintaxe:
valor_se_verdadeiro if condicao else valor_se_falso
Exemplo Prático:
idade = 19
status = "Maior de idade" if idade >= 18 else "Menor de idade"
print(status)  # Saída: Maior de idade

saldo = 500
pode_sacar = 100
mensagem_saque = "Saque autorizado" if saldo >= pode_sacar else "Saldo insuficiente"
print(mensagem_saque) # Saída: Saque autorizado

Isso é útil para atribuições simples baseadas em uma condição.

Lembre-se da Indentação!

A indentação (o recuo do código com espaços) é fundamental em Python. É ela que define os blocos de código que pertencem a cada comando if, elif, ou else. Erros de indentação são uma causa comum de problemas em Python.

Use sempre um número consistente de espaços (a convenção é usar 4 espaços) para indentar seu código dentro de cada bloco.
